# AI Maze Solver - Reinforcement Learning & A* Pathfinding

## Overview

This project is a Unity-based maze game that demonstrates the use of two artificial intelligence techniques:

* **Reinforcement Learning (RL)** for solving mazes.
* **A* (A-Star) Pathfinding Algorithm** for enemy movement and navigation.

The project was developed to explore and compare different AI approaches within a game environment. The reinforcement learning agent learns how to navigate the maze through trial and error, while enemy characters use the A* algorithm to find the shortest path to their target.

## Features

### Reinforcement Learning Maze Solver

* AI agent learns to navigate through a maze.
* Uses rewards and penalties to improve decision-making.
* Demonstrates autonomous learning without predefined routes.
* Training occurs through repeated episodes and environment interaction.

### A* Enemy Pathfinding

* Enemies use the A* algorithm to calculate optimal routes.
* Dynamically finds the shortest path around obstacles.
* Efficient navigation through complex maze layouts.
* Real-time path recalculation when required.

### Unity Implementation

* Built using the Unity Game Engine.
* Grid-based maze environment.
* Visual representation of AI decision-making and movement.
* Modular scripts for easy expansion and experimentation.
